# Description
I want to have separate screens for different areas of the game. They should be accessed using hotkeys. The default starting screen is castle screen.
Available screens:
* Castle - hotkey 1
* Training grounds - hotkey 2
* Bridges with farmlands - hotkey 3
* well and outside - hotkey 4
* personal domain - hotkey 5

On top of every screen, the following information should be desplayed:
* Current player in color of the player
* Victory Points for each player
* influence for each player
* round number
