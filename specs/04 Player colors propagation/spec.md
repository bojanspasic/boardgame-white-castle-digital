# Description
Player specific details should be shown with their respective colors chosen in the main menu.

# Status
COMPLETED

# Acceptance criteria
* When AI plays, the overlay message ([in AiThinkingOverlay.cs](/src/BoardWC.Console.UI/AiThinkingOverlay.cs)) is shown in the color of the AI player selected in the [main menu](/src/BoardWC.Console.UI/MainMenu.cs).
* Players are color-coded as follows: Player 1 - blue, Player 2 - red, Player 3 - green, Player 4 - yellow.
* Color codes are available for the whole Console UI so it can be reused on other places.
